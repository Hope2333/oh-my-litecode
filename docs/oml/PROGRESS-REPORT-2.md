## 📝 决策记录

### 决策 1: Grep-App 双模式支持

**决策**: 支持 stdio 和 HTTP 两种模式

**理由**:
- stdio: 本地 npx 执行，无需额外服务
- HTTP: 远程服务，适合集中部署

**状态**: ✅ 已实施

---

### 决策 2: Build Agent Makefile 集成

**决策**: 直接调用现有 Makefile，不重复实现

**理由**:
- 现有 Makefile 已完整
- 避免代码重复
- 保持向后兼容

**状态**: ✅ 已实施

---

### 决策 3: Phase 2.5 范围

**决策**: Phase 2.5 聚焦 Grep-App + Build/Plan/Reviewer

**理由**:
- Grep-App 是 Scout 依赖
