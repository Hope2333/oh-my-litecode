# Oh-My-Litecode Glossary

## Project Names

| Abbrev | Full Name | Description |
|--------|-----------|-------------|
| OML | Oh-My-Litecode | Main orchestrator project |
| OCT | OpenCode(-on)-Termux | OpenCode builds for Termux |
| OMG | omgemini | Gemini CLI integration (future) |
| OMF | omforge | ForgeCode integration (future) |
| OMA | omaider | Aider integration (future) |

## Technical Terms

| Term | Definition |
|------|------------|
| grun | glibc-runner, executes glibc binaries on Termux |
| bunfs | Bun virtual filesystem for embedded assets |
| PIE | Position Independent Executable |
| BUNWRAP1 | Marker in bun-termux-loader output |
| staged build | Build without npm global postinstall |

## Package Naming

| Pattern | Example |
|---------|---------|
| Standard | `opencode-1.1.65-1.pacman.tar.xz` |
| Debug | `opencode-debug-1.1.65-1.pacman.tar.xz` |
