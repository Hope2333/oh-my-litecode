## 📝 决策记录

### 决策 1: Worker 池默认配置

**决策**: 默认并发数 = min(4, CPU 核心数)

**理由**:
- 平衡性能与资源占用
- 适配不同硬件配置
- Termux 友好

**状态**: ✅ 已实施

---

### 决策 2: Hooks 默认禁用

**决策**: Hooks 默认禁用，需显式启用

**理由**:
- 向后兼容
- 避免性能开销
- 用户可选择性启用

**状态**: ✅ 已实施

---

### 决策 3: Session 存储路径

**决策**: ~/.oml/sessions/<date>/<session-id>/

**理由**:
- 按日期分组，便于管理
- 避免单目录文件过多
- 便于清理过期会话

**状态**: ✅ 已实施
