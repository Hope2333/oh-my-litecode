### 1.1.1 文件树生成算法

**实施步骤**:
1. 遍历目标目录
2. 构建文件系统树
3. 计算节点统计信息
4. 生成 ASCII/JSON 输出

**代码示例**:
```bash
oml_scout_generate_tree() {
    local target="$1"
    local max_depth="${2:-3}"
    
    find "$target" -maxdepth "$max_depth" -type f | \
    sort | \
    awk -F/ '{
        for (i=1; i<=NF; i++) {
            printf "%s", (i==NF ? $i : $i "/")
        }
        print ""
    }'
}
```

### 2.2.1 resolve-library-id 调用封装

**实施步骤**:
1. 接收库名称和查询
2. 调用 Context7 MCP
3. 解析返回结果
4. 缓存库 ID 映射

**代码示例**:
```bash
librarian_resolve_library() {
    local library_name="$1"
    local query="$2"
    
    # 调用 MCP 工具
    local result
    result=$(oml qwen -p "Call tool mcp__context7__resolve-library-id with {\"libraryName\":\"${library_name}\",\"query\":\"${query}\"}")
    
    # 解析结果
    echo "$result" | python3 -c "
import json, sys
data = json.load(sys.stdin)
print(data.get('libraryId', ''))
"
}
```
