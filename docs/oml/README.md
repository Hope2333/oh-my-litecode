### 文档命名规范

- **大写**: 主文档 (README, QUICKSTART)
- **大写 + 连字符**: 报告类 (EXPLORATION-ASSESSMENT)
- **大写 + 连字符**: 任务类 (PHASE2-TODOS)
- **小写 + 连字符**: 子目录文件

### 文档版本控制

- 所有文档在文件头标注版本号
- 重大更新升级版本号
- 文档变更在末尾标注"最后更新"
