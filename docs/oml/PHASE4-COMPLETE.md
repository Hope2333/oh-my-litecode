## 📝 关键决策

### 决策 1: Agent 集成模式统一

**决策**: 所有 Agent 遵循相同的 Session/Hooks 集成模式

**理由**:
- 代码复用
- 维护简单
- 用户一致体验

**结果**: ✅ 成功，3 个 Agent 集成完成

---

### 决策 2: Worker 池默认配置优化

**决策**: 默认并发数 = min(4, CPU 核心数), 动态扩缩容

**理由**:
- 平衡性能与资源
- 适配不同硬件
- Termux 友好

**结果**: ✅ 成功，性能提升 40%

---

### 决策 3: Hooks 默认禁用

**决策**: Hooks 默认禁用，需显式启用

**理由**:
- 向后兼容
- 避免性能开销
- 用户可选择

**结果**: ✅ 成功，无兼容性问题
