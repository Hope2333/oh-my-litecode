# Codex CLI 定制和扩展

本目录包含针对 OpenAI Codex CLI 的专门定制、扩展和功能移植。

## 目录结构

```
codex/
├── extensions/          # Codex 扩展
├── plugins/             # MCP 服务器和插件
├── commands/            # 自定义命令
├── scripts/             # 辅助脚本
└── docs/                # 文档
```

## Codex CLI 快速开始

### 安装
```bash
npm install -g @openai/codex
```

### 配置
```bash
export OPENAI_API_KEY=your-key
```
